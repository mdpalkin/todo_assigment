# Task Tracker 

Программа для управления списками задач

## Возможности

1. Добавление задач
2. Удаление задач
3. Смена названия задачи путем двойного клика на название
4. Перетаскивание задач как в рамках одной колонки, так и между столбцами

## Установка зависимостей

```
pnpm install
```

## Запуск проекта 

```
pnpm run dev
```
